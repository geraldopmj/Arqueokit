about=Arqueokit is a QGIS plugin designed to support archaeological workflows with processing tools for public data download, reporting, charts, raster analysis, machine learning and web map export.
  Some algorithms require external Python packages in the QGIS Python environment, including reportlab, pandas, matplotlib, seaborn, rasterio, scikit-learn, scipy, joblib, scikit-image, Pillow, folium, flet and flet-map.
  Some tools also depend on internet access and on the availability of third-party public services such as FUNAI and IPHAN servers.
  Users should install the required Python packages in the QGIS environment before running the algorithms that depend on them.
